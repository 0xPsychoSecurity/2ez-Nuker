# 2ez-Nuker
**Discord Server Nuker. Uses a bot with admin permissions to instantly destroy a server.**
<img width="1213" height="726" alt="image" src="https://github.com/user-attachments/assets/b14f59ec-0bf4-46df-a705-11844dbb6547" />

Changeable Values:
- Server Name
- Server Icon
- Channel Name 
- Role Name
- Text after @everyone



Must have a Discord Bot Token, and you have to invite the bot to the server with admin permissions, the program will generate a link to invite it to the server with whatever bot token you provide.
